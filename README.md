# FKG List

A simple character list for [Flower Knight Girl](http://pc-play.games.dmm.com/play/flower/) using [React](http://reactjs.org/).

##### Folders

* `client`: Main project.
* `crawler`: Data crawler/generator for client.
* The rest are not important.
